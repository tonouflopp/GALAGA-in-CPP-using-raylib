Galaga Clone

A Galaga-style arcade shooter built in C++ with Raylib built With
C++ and Raylib, compiled with MinGW / MSYS2

Requisites:
MSYS2 with MinGW-w64
Raylib installed and linked
VS Code (recommended) or any C++ IDE

Running

Make sure Raylib is installed and accessible to your compiler.
Open the project folder in VS Code and press F5 to build and run (uses the included tasks.json / launch.json configuration).


Controls

Action Keys -> Move ship
Space -> Fire laser
Esc -> Quit
